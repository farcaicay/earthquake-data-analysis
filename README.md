# Seismic Activity Analysis in Peru (1960–2025)

## 📊 Project Overview
This project analyzes earthquake data in Peru to identify seismic patterns and risk zones.

## 🔧 Tools Used
- Python (Pandas, GeoPandas)
- Power BI
- Shapefiles (Geospatial Data)

## 📈 Key Insights
- Most earthquakes are moderate (5–6 magnitude)
- Coastal regions show highest activity
- Low correlation between depth and magnitude
