# Analysis of seismic data in Peru

## Project Description
This project analyzes historical earthquake data to identify patterns in magnitude, depth, and geographic distribution.

## Dataset
The dataset includes:
- Date and time
- Latitude and longitude
- Magnitude
- Depth
- Country and region

## Tools
Python
Pandas
Matplotlib
Power BI

## Goal
Explore earthquake activity and visualize seismic patterns in South America.
